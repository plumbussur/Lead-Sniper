# Анализатор российских компаний, использующих CAT-системы

Инструмент для сбора и анализа данных о российских компаниях, которые используют CAT-системы (Computer-Aided Translation) и имеют выручку от 100 млн рублей.

## Подход

### Как искали источники данных

1. **Rusprofile.ru** - основной источник финансовых данных и ИНН компаний
2. **Отраслевые каталоги** - специализированные справочники компаний локализационной отрасли
3. **Анализ сайтов компаний** - парсинг корпоративных сайтов для поиска признаков CAT

### Логика определения CAT-систем

Компания считается использующей CAT-системы, если найдены следующие признаки:

**Ключевые слова:**
- translation memory, tm, терминология, память переводов
- cat tool, cat-система, tms, translation management
- локализация, localization

**Продукты:**
- Trados, MemoQ, Across, WordFast, Memsource, Smartcat
- Phrase, Lokalise, Crowdin

**Фразы:**
- "система управления переводами"
- "компьютерная поддержка переводов"
- "платформа локализации"

### Подтверждение признаков

Доказательство наличия CAT-системы формируется из:
- Упоминаний в описании деятельности компании
- Названий продуктов на корпоративном сайте
- Описания технологических процессов

### Проверка выручки

- Минимальный порог: 100 млн рублей
- Проверка актуальности данных (последний доступный год)
- Валидация разумности значений


## Роль LLM

В данном проекте LLM не использовались, так как задача решалась классическими методами обработки текста и поиска ключевых слов. Однако в будущем планируется:

- Генерация описаний компаний на основе открытых данных
- Улучшение классификации "есть CAT/нет CAT"
- Автоматическое извлечение информации о продуктах

## Установка и запуск

### Требования
- Python 3.8+
- pip

### Установка
```bash
# Клонирование репозитория
git clone <repository-url>
cd cat-companies-analyzer

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Полный сбор и анализ данных
python src/main.py --collect

# Анализ существующих данных
python src/main.py --analyze

# Сбор из конкретных источников
python src/main.py --collect --sources rusprofile catalog

# Сохранение в другой файл
python src/main.py --collect --output results/my_companies.csv
```

## Результат

Создается файл data/companies.csv со следующими полями:
- `inn`: ИНН компании
- `name`: Название компании
- `revenue`: Финансовые данные, если доступны
- `site`: Сайт компании, если доступно 
- `cat_evidence`: Доказательство наличия CAT-системы
- `source`: Источник данных
- `cat_product`: Классификация продуктов
- `employees`: Количество сотрудников
- `okved_main`: Основной ОКВЭД
- `country`: Страна
    
    

## Архитектура
```
src/data_collectors/ - сбор данных из различных источников
src/processors/ - обработка и фильтрация данных
src/utils/ - вспомогательные утилиты
config/ - конфигурация и настройки
```
